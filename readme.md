## SSMS com Leitores de tela
O SSMS apresenta boa acessibilidade para uso com leitores de tela, sendo preciso dominar alguns atalhos importantes.

### Atalhos importantes

|Descrição|Atalho|
|--|--|
|Executar consulta|F5|
|Mover para explorador de objetos|F8|
